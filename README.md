# Customer Churn Prediction

## About the Project
This project focuses on predicting whether a customer is likely to leave a company (churn). Customer retention is an important problem for many businesses, so being able to identify potential churners can help improve decision-making.

I built this project as part of my data science learning journey to apply machine learning techniques to a real-world problem.

## What I Did
- Cleaned and prepared the data for modeling  
- Handled class imbalance using RandomOverSampler  
- Trained multiple machine learning models  
- Compared model performance  

## Models Used
- Logistic Regression  
- K-Nearest Neighbors  
- Random Forest  
- Voting Classifier
